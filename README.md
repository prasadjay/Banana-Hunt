Banana-Hunt
===========

A 2D sample created to check Cocos2D framework. Hobby project. Not Completed. 

What's already in?

1) sprites
2) collisions
3) custom input system algorithm ( without using sneakyinputs and etc)
3) Velocity based jumping and action algorithm
